#+title: Skills
#+author: Giovanni Crisalfi

My AI agent skills. Written in [[https://orgmode.org][Org-mode]] and exported with [[https://github.com/gicrisf/ox-skills][ox-skills]].

* Workflow

1. Edit =SKILLS.org=
2. Run =M-x ox-skills-export-wim-to-md= (=C-c C-e s a=, or =SPC m e s a= in evil)

Hand-written =.md= files in =skills/= are left untouched (you can mix exported and manual skills freely).

** Creating a new skill

Load the [[file:skills/meta/org-skill-maker/SKILL.md][org-skill-maker]] skill to start immediately! It asks what you need, adds the right entry to =SKILLS.org= with ~EXPORT_SKILL_*~ properties, and exports it.

Install first (see below) to make it available in your environment, or just tell your agent:

#+begin_quote
Load the org-skill-maker skill and create a skill for <your task>
#+end_quote

* Installation

#+begin_src sh
git clone git@github.com:gicrisf/skills.git ~/dev/skills
#+end_src

** Batch (recommended)

Requires [[https://www.gnu.org/software/stow/][GNU Stow]].

#+begin_src sh
cd ~/dev/skills && stow -t ~ links
#+end_src

Symlinks =.claude/skills=, =.opencode/skills=, =.agents/skills= and anything in =links/=.

** Single symlink

#+begin_src sh
ln -sfn ~/dev/skills/skills ~/.agents/skills
#+end_src

To undo:

#+begin_src sh
unlink ~/.agents/skills
#+end_src
